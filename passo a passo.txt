Passos sobre o github com Django:

Primeiro, entrei na pasta do projeto e liguei o ambiente virtual venv (venv\Scripts\activate)
Lembrando que pra ativar ele, primeiro é preciso criar o ambiente: crie a pasta e ao entrar nela, digite "python -m venv venv" sem as aspas. O segundo venv é o nome do arquivo do ambiente virtual, pode colocar qualquer um mas é interessante chamar de venv.

Depois de ligar o venv, setei algumas configurações de identificação:
git config --global user.name "Samuel Ágap da Silva Santos"
git config --global user.email "samuelagap3@gmail.com"
git config --global init.defaultBranch main
git init

Depois disso, ele criou o arquivo git na minha pasta. Dei enter pra tudo até criar.
Fui no local da chave pública criada "Your public key...." e abri num bloco de nota. Copiei a chave, fui no github > settings > SSH and GPG keys > Criei nova chave SSH > copiei a chave lá e pronto.

Depois, voltei pro repositório, alterei o setup de HTTPS pra SSH e copiei o link SSH.
No cmd do projeto, digitei: git remote add origin (link SSH copiado)
Pra saber os link anexados, digita git remote -v
